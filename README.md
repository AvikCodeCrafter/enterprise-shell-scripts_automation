## Overview

Enterprise Shell Scripts Automation is a curated collection of production-ready Bash scripts, DevOps utilities, and Site Reliability Engineering (SRE) automation tools.

The goal of this repository is to simplify day-to-day infrastructure operations, cloud management, Kubernetes administration, monitoring, deployments, and security tasks using reusable shell scripts.

This repository serves as my personal DevOps automation toolkit and is continuously enhanced with scripts developed from real-world SRE experience.
